# 🦇 Bat Signal

Um aplicativo mobile simples feito com **React Native e TypeScript** chamado *Bat Signal*.

Este projeto foi criado para aprender a desenvolver interfaces e interações com React Native, usando botões, entradas de texto e navegação básica.

---

## 📌 Tecnologias

O projeto foi desenvolvido com:

- **React Native**
- **TypeScript**
- **Expo / React Native CLI** (configuração base do projeto)
- 📱 Componentes customizados como botões e inputs

---

## 🚀 Como usar

1. **Clone o repositório:**

```bash
   git clone https://github.com/xCaio/bat-signal.git
```

2. **Instale as dependências:**
``` bash
   npm install
   # ou
   yarn
```


3. Inicie o app:
``` bash
npm start
# ou
yarn start
```
